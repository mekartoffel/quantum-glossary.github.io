# Coupling Graph
{: .fs-9 }
A graphical representation of interactions between components in a specific hardware layout.
{: .fs-6 .fw-300 }

Quantum
{: .label .label-green}

## Full Definition
A graph \\(G={V,E} \\) where:
- \\(V\\) represent qudits in the hardware.
- \\(E\\) represent a physically possible coupling between two qudits.
